DragonChanges is a mod for all the changes that aren't bug fixes that I could include in DragonFixes.

Current features include
* Adding the Hippogriff pet to other animal companion selections
* Adding the Nightmare pet from [Microscopic Content Expansion](https://www.nexusmods.com/pathfinderwrathoftherighteous/mods/506) to other animal companion selections (This requires MCE to be installed)
* Adds a super secret option for buffing [Expanded Content](https://www.nexusmods.com/pathfinderwrathoftherighteous/mods/377)'s drake companion's initial stats. Disabled by default
* Add new feature [Powerful Throw](https://www.d20pfsrd.com/magic/variant-magic-rules/akashic-magic/feats/powerful-throw-combat/) - requested by Rafanta89 on Discord
* Add new mount (from an existing model in game), the undead horse mount. -requested by Scanix on Discord
* Adds a new Unicorn mount (enabled by default) - mountable, looks mostly fine?
* Adds a new Griffon mount (enabled by default) - mountable, just tposes, will eventually do mount points for it
* Adds new feature [Powerful Wings](https://aonprd.com/FeatDisplay.aspx?ItemName=Powerful%20Wings) (enabled by default)
* Adds my meme class Redditor (disabled by default)
* New Rogue archetype - [Swordmaster](https://www.d20pfsrd.com/races/other-races/featured-races/arg-tengu/swordmaster-rogue-tengu/)
* New background, Cursed (Lycanthropy) for Rafatan89
* New Mythic feats, Auto<metamagic> which gives you an ability to automatically apply the selected Metamagic to your spells
* Adds the Heavenfire Ray spell, which is a good aligned version of Hellfire Ray. Added to several spell lists, at spell level 6
* Adds the [HellBall](https://www.d20srd.org/srd/epic/spells/hellball.htm) spell, which you get from a Mythic Ability. Because someone asked and I thought it was funny. Also it does 20d6 instead of 10d6
* New Mythic feat,  Grave Singer Plus - Adds 1 crit range and 1 crit multiplier. Yup, that's it. Extremely OP
* [Storval's Thunder](https://aonprd.com/MagicWeaponsDisplay.aspx?ItemName=Storval%27s%20Thunder%20(Shoanti)) - an Earthbreaker with a set bonus with Storval's Fang
* [Storval's Fang](https://www.aonprd.com/MagicArmorDisplay.aspx?ItemName=Storval%27s%20Fang) - a spiked light shield with a set bonus with Storval's Thunder
* Adds the Heavenfire Ray scroll, and adds it to my new vendor
* Adds a new ring, which adds CON bonus to your crit range, see my new vendor
* Add the [Radiant Prism](https://2e.aonprd.com/Deities.aspx?ID=380) deity
* Super Dexterous Shot, a new Mythic ability that grants DEX bonus to bows
* New necklace that grants STR bonus to DR, available in my Anevia vendor
* Adds KM weapon Ruin to Anevia vendor so you can actually use it
* Adds KM weapon Serpent Prince to Anevia vendor so you can actually use it
* New mythic - Abundant All Is Darkness, works the same as the other abundants
* New ring - King's Ring, gives the user the ability to use the Cavalier's ability For the King once per battle
* New ring - Ring of Arcane Success, lowers arcane spell failure by 25
* New bracers - Bracers of Cleaving Rafanat, gives ranged cleave in 30ft radius
* Insightful Contemplation, the Court Poet ability, now grants +2 WIS as well, mythic also increases this bonus (requested by Cyrix)
* New archetype - Juggernaut (Barbarian), gets DR/- from CON bonus. Inspired by Rafanat on Discord
* Adds some KM bastard swords to Anevia
* BuildsEnjoyer meme class , a combination of Inciter / Sable Marine / Drunken Monk with Deceiver spell book
* New spell, Death Ward Communal. All the same spell lists as regular Death Ward, just level 7 and communal. Has a scroll version in my vendor
* New mod compatibility option. For those using AlterAsc's Microscopic Content Expansion, this will alter the spell Deadly Juggernaut to have it's DR stack. Disabled by default
* Start of new item sets. Started implementing various [Ioun Stones](https://www.d20pfsrd.com/magic-items/wondrous-items/h-l/ioun-stones/), you can buy them at my vendor.
* Add new feat, Exotic Weapon Master. Gives proficiency with all Exotic weapons. Requested by nicksnax on Discord
* New meme feat (disabled by default) called My Thic Feat. Gives +2 CON. Kab on Discord misread "MythicFeat" as "My Thic Feat" and thus, this was borne.
* Add new setting, buffs the Trickster Stat Focus (Speed)/(HP) to not be absolute garbage
* New spell, Communal Sense Vitals and scroll
